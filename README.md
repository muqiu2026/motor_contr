# 电机控制项目

## 项目简介

本项目使用PY32F002芯片实现有刷电机的控制，包括以下功能：

- 编码器检测电机运行方向和速度
- 双PWM控制电机
- LED指示灯
- 按键控制（正转/反转切换）
- 缓起缓停功能
- 记录电机转动圈数和运行方向
- 串口控制功能

## 硬件连接

| 功能 | 引脚 |
|------|------|
| LED | PA0 |
| 按键 | PA1 |
| 编码器A相 | PA2 |
| 编码器B相 | PA3 |
| 电机PWM1 | PA4 |
| 电机PWM2 | PA5 |
| 串口TX | PA9 |
| 串口RX | PA10 |

## 软件功能

1. **按键控制**：按一下正转，再按一下反转，正反转切换时会先减速停止后再运行
2. **缓起缓停**：电机启动和停止过程中会进行缓起缓停，以保护电机和机械结构
3. **编码器检测**：实时检测电机运行方向和速度，记录电机转动圈数
4. **串口控制**：通过串口协议控制电机的运行，包括正转、反转、停止、运行圈数、读取当前电机位置等

## 串口协议

详细的串口协议请参考 `serial_protocol.md` 文件。

## 编译与烧录

1. 使用Keil MDK或其他支持PY32F002的IDE打开项目
2. 编译生成hex文件
3. 使用ST-Link或其他调试器烧录到PY32F002芯片

## 注意事项

1. 电机驱动电路需要根据实际电机参数进行设计
2. 编码器的PPR（每转脉冲数）需要根据实际编码器进行修改
3. 缓起缓停参数可以根据实际需求进行调整
4. 串口波特率默认为115200，可根据需要修改
