# 电机控制串口协议

## 1. 协议概述

本协议用于通过串口控制电机的运行，包括正转、反转、停止、运行指定圈数以及读取电机状态等功能。

## 2. 通信参数

- 波特率：115200
- 数据位：8位
- 停止位：1位
- 校验位：无
- 流控制：无

## 3. 命令格式

所有命令均为文本格式，以回车换行符（\r\n）结束。

## 4. 命令列表

### 4.1 正转命令

**命令格式**：`FORWARD`

**功能**：控制电机正转

**响应**：`Motor forward\r\n`

### 4.2 反转命令

**命令格式**：`BACKWARD`

**功能**：控制电机反转

**响应**：`Motor backward\r\n`

### 4.3 停止命令

**命令格式**：`STOP`

**功能**：控制电机制动停止

**响应**：`Motor stop\r\n`

### 4.4 读取位置命令

**命令格式**：`POSITION`

**功能**：读取电机当前位置（编码器计数）

**响应**：`Motor position: [位置值]\r\n`

### 4.5 读取方向命令

**命令格式**：`DIRECTION`

**功能**：读取电机当前运行方向

**响应**：
- 正转：`Motor direction: forward\r\n`
- 反转：`Motor direction: backward\r\n`
- 停止：`Motor direction: stop\r\n`

### 4.6 运行指定圈数命令

**命令格式**：`RUN:[圈数],[方向]`

**参数**：
- `[圈数]`：要运行的圈数（正整数）
- `[方向]`：运行方向（1表示正转，-1表示反转）

**功能**：控制电机运行指定的圈数后停止

**响应**：
- 命令执行中：无响应
- 命令执行完成：`Motor run completed\r\n`
- 方向参数错误：`Invalid direction\r\n`

## 5. 错误处理

- 未知命令：`Unknown command\r\n`
- 命令格式错误：根据具体错误返回相应提示

## 6. 示例

### 6.1 控制电机正转

**发送**：`FORWARD\r\n`
**接收**：`Motor forward\r\n`

### 6.2 控制电机反转

**发送**：`BACKWARD\r\n`
**接收**：`Motor backward\r\n`

### 6.3 停止电机

**发送**：`STOP\r\n`
**接收**：`Motor stop\r\n`

### 6.4 读取电机位置

**发送**：`POSITION\r\n`
**接收**：`Motor position: 1234\r\n`

### 6.5 读取电机方向

**发送**：`DIRECTION\r\n`
**接收**：`Motor direction: forward\r\n`

### 6.6 运行指定圈数

**发送**：`RUN:5,1\r\n`
**接收**：`Motor run completed\r\n`

## 7. 注意事项

1. 命令区分大小写，建议使用大写字母发送命令
2. 命令必须以回车换行符结束
3. 运行指定圈数命令执行过程中，系统会阻塞直到完成指定圈数
4. 电机启动和停止过程中会进行缓起缓停，以保护电机和机械结构
5. 正反转切换时，系统会先减速停止，然后再反向启动
